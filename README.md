# Transliteration Accuracy Testing - Playwright Automation

## 📌 Project Overview

This project is developed for **IT3040 – ITPM Assignment 1 (Option 1)**.

The objective of this project is to:

* Evaluate the accuracy of a Sinhala transliteration system
* Identify incorrect conversions from Singlish to Sinhala
* Automate testing using Playwright

System under test:
https://www.pixelssuite.com/chat-translator

---

## 🛠 Technologies Used

* Python 3.11 / 3.12
* Playwright
* OpenPyXL
* Google Chrome / Chromium

## ▶️ How to Run the Script

Run the following command:

python test_automation.py --excel "Assignment 1 - Test cases.xlsx" --url "https://www.pixelssuite.com/chat-translator" --wait-ms 5000 --type-delay-ms 80 --slow-mo-ms 200 --save-every 1 --keep-open


---

## 📊 How the Automation Works

* Reads test cases from the Excel file
* Inputs Singlish text into the web application
* Captures Sinhala output from the UI
* Compares it with expected output
* Writes results into Excel:

  * Actual Output
  * Status (PASS / FAIL)

---

## 📌 Important Notes

* Do NOT manually fill:

  * "Actual Output"
  * "Status"
    (These are automatically generated by the script)
* Make sure the Excel file is in the same folder as the script
* Internet connection is required to run the tests
* If the website is down, tests will fail (this is expected behavior)

---

## 📊 Output

After running the script:
